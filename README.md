# esphome-irrigation-controller
Irrigation controller based on ESPHome ([Sprinkler controller](https://esphome.io/components/sprinkler.html)) and ESP32 MCU.

## Hardware for this project
* [8 channel ESP32 relay board](https://a.aliexpress.com/_EzzVPp8)
* [Dallas temperature sensor (DS18B20)](https://a.aliexpress.com/_EzwiXpU)
* [12-24V ball valve](https://a.aliexpress.com/_ExImKO2)
* [AC/DC converter](https://a.aliexpress.com/_EGvSTsi)
* [SH1106 1.3" display](https://a.aliexpress.com/_EuUBVeQ)
